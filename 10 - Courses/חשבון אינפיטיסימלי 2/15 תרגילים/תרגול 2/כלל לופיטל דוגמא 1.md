# כלל לופיטל דוגמא 1

## השאלה
האם קיים ה[[הגדרה - גבול של פונקציה|גבול]] אם כן, מה ערכו?
$$
\lim_{ x \to 0 } \frac{\sin x}{\arctan x}
$$
שני הפונקציות ערכן 0 בנקודת הגבול אבל לא ברור כל כך מהו היחס ביניהן, לכן 
## פתרון

נבקש להפעיל את **כלל לופיטל** ניזכר בתנאיו:

![[4.1.1 כלל לופיטל לאפס חלקי אפס#תנאי המשפט]]

נסמן 
$$
	f(x) = \sin x \quad g(x) = \arctan x
$$
ואכן שתיהן רציפות בסביבה  $U$ של $0$ בפרט בכל $\mathbb{R}$ של $x$ ולכן מתקיים
$$
	\lim_{ x \to 0 } f(x) = \lim_{ x \to 0 } g(x) = f(0) = g(0) =0
$$
כלומר התנאי הראשון מתקיים.
התנאי השני מתקיים מאחר והפונקציות גזירות בכל $\mathbb{R}$
בשביל לבדוק את התנאי השני נגזור 
$$
	(\arctan x)' = \frac{1}{1+x^2}
$$
ואכן 
$$
	g'(x)\neq 0
$$
לכל $x\in \mathbb{R}$ ובפרט ב $U$ כלשהי. התנאי הרביעי הוא גם הפתרון במקרה שלנו שכן 
$$
	\lim_{ x \to 0 } \frac{f'(x)}{g'(x)} = \lim_{ x \to 0 } \frac{\cos x}{\frac{1}{1+x^2}} = \lim_{ x \to 0 } \cos (x)(1+x^2)  = 1
$$
ולכן מקיום הכלל הרביעי **כלל לופיטל מתקיים** ו-
$$
\lim_{ x \to 0 } \frac{\sin x}{\arctan x} = 1
$$
