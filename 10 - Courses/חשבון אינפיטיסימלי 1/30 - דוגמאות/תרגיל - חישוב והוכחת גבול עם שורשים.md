# תרגיל: חישוב והוכחת גבול (כפל בצמוד)

**המקור:** שאלה 2 ממבחן/תרגיל.

**השאלה:**
נתונה הסדרה המוגדרת על ידי:
$$a_n = \sqrt{n^2 + 2n} - n$$
א. חשבו את הגבול $\lim_{n \to \infty} a_n$ בעזרת אריתמטיקה של גבולות.
ב. הוכיחו את הגבול שמצאתם ישירות לפי הגדרת הגבול ($\epsilon - N$).

---

### פתרון סעיף א': חישוב הגבול
במבט ראשון, מדובר בגבול מהצורה "$\infty - \infty$" (לא מוגדר).
הטכניקה לפתרון: **כפל וחילוק בצמוד**.

$$a_n = \frac{(\sqrt{n^2 + 2n} - n)(\sqrt{n^2 + 2n} + n)}{\sqrt{n^2 + 2n} + n}$$

נפתח סוגריים במונה לפי $(a-b)(a+b) = a^2 - b^2$:
$$a_n = \frac{(n^2 + 2n) - n^2}{\sqrt{n^2 + 2n} + n} = \frac{2n}{\sqrt{n^2 + 2n} + n}$$

נוציא גורם משותף $n$ מהמכנה (נזכור ש-$\sqrt{n^2} = n$ עבור $n>0$):
$$a_n = \frac{2n}{n\sqrt{1 + \frac{2}{n}} + n} = \frac{2n}{n(\sqrt{1 + \frac{2}{n}} + 1)} = \frac{2}{\sqrt{1 + \frac{2}{n}} + 1}$$

נשאיף את $n \to \infty$. הביטוי $\frac{2}{n}$ שואף ל-0, ולכן:
$$\lim_{n \to \infty} a_n = \frac{2}{\sqrt{1+0} + 1} = \frac{2}{2} = 1$$

---

### פתרון סעיף ב': הוכחה לפי הגדרה
צריך להוכיח: לכל $\epsilon > 0$ קיים $N$ כך שלכל $n > N$ מתקיים $|a_n - 1| < \epsilon$.

**1. טיוטה (מציאת הקשר בין $n$ ל-$\epsilon$):**
נפשט את הביטוי שבתוך הערך המוחלט:
$$|a_n - 1| = |\sqrt{n^2 + 2n} - n - 1| = |\sqrt{n^2 + 2n} - (n + 1)|$$
נשים לב ש-$(n+1) = \sqrt{(n+1)^2} = \sqrt{n^2 + 2n + 1}$.
לכן השורש המקורי קטן מ-$n+1$, והביטוי בתוך הערך המוחלט שלילי. נהפוך סדר:
$$= (n+1) - \sqrt{n^2 + 2n}$$
נכפול בצמוד של הביטוי הזה כדי להיפטר מההפרש:
$$= \frac{((n+1) - \sqrt{n^2 + 2n})((n+1) + \sqrt{n^2 + 2n})}{(n+1) + \sqrt{n^2 + 2n}}$$
$$= \frac{(n+1)^2 - (n^2 + 2n)}{n + 1 + \sqrt{n^2 + 2n}} = \frac{1}{n + 1 + \sqrt{n^2 + 2n}}$$
כעת נבצע **הגדלה** של הביטוי (נמצא חסם עליון פשוט יותר) על ידי הקטנת המכנה:
$$\frac{1}{n + 1 + \sqrt{n^2 + 2n}} < \frac{1}{n}$$
דרישה: $\frac{1}{n} < \epsilon \implies n > \frac{1}{\epsilon}$.

**2. ההוכחה הפורמלית:**
יהי $\epsilon > 0$.
נבחר $N$ טבעי המקיים $N > \frac{1}{\epsilon}$.
לכל $n > N$ מתקיים:
$$|a_n - 1| = (n+1) - \sqrt{n^2 + 2n} = \frac{1}{n + 1 + \sqrt{n^2 + 2n}} < \frac{1}{n} < \frac{1}{N} < \epsilon$$
לכן הגבול הוא 1.
