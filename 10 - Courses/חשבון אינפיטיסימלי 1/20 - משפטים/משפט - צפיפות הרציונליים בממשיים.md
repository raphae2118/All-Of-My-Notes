# משפט: צפיפות הרציונליים בממשיים

**המשפט:** הקבוצה $\mathbb{Q}$ צפופה ב-$\mathbb{R}$.
כלומר: בין כל שני מספרים ממשיים $x < y$, קיים מספר רציונלי $q \in \mathbb{Q}$ כך ש:
$$x < q < y$$

---

**הוכחה (בנייה לפי מקרים):**
נפריד למקרים לפי המיקום של $x$ ו-$y$ על הציר:

### מקרה א': המספרים משני צדי האפס ($x < 0 < y$)
זהו המקרה הטריוויאלי.
נבחר פשוט $q = 0$. מכיוון ש-$0 \in \mathbb{Q}$, מצאנו מספר רציונלי ביניהם.

### מקרה ב': שני המספרים אי-שליליים ($0 \le x < y$)
זוהי ליבת ההוכחה.
1. **צעד האפסילון:** נסמן $\epsilon = y - x$ (המרחק בין המספרים).
   לפי המסקנה הראשונה של [[משפט - תכונת ארכימדס]], קיים מספר טבעי $n \in \mathbb{N}$ כך שגודל הצעד $\frac{1}{n}$ קטן מהמרחק ביניהם:
   $$\frac{1}{n} < y - x$$

2. **מציאת המונה ($m$):**
   נגדיר קבוצה $A$ של כל המספרים הטבעיים שמכפלתם ב-$\frac{1}{n}$ כבר "עוקפת" את $x$:
   $$A = \{ m \in \mathbb{N} \mid x < m \cdot \frac{1}{n} \}$$
   * קבוצה זו **אינה ריקה** (לפי ארכימדס, המכפלות של $\frac{1}{n}$ שואפות לאינסוף ולכן יעברו את $x$).
   * לכן, לפי **עקרון הסדר הטוב** של המספרים הטבעיים, קיים לקבוצה איבר מינימלי. נסמן אותו $m_0 = \min(A)$.

3. **בניית המספר:**
   נסמן $q = \frac{m_0}{n}$. ברור כי $q \in \mathbb{Q}$.

4. **הוכחה שהוא מתאים:**
   * **מצד אחד:** $x < q$ (נובע ישירות מההגדרה ש-$m_0 \in A$).
   * **מצד שני:** נראה ש-$q < y$.
     נביט ב-$m_0 - 1$. מכיוון ש-$m_0$ הוא המינימום, האיבר שלפניו **לא** בקבוצה $A$. לכן:
     $$\frac{m_0 - 1}{n} \le x$$
     נוסיף $\frac{1}{n}$ לשני האגפים כדי לקבל את $q$:
     $$q = \frac{m_0}{n} = \frac{m_0 - 1}{n} + \frac{1}{n} \le x + \frac{1}{n}$$
     אבל בחרנו את $n$ כך ש-$\frac{1}{n} < y - x$, ולכן:
     $$q \le x + \frac{1}{n} < x + (y - x) = y$$
   **מסקנה:** קיבלנו $x < q < y$.

### מקרה ג': שני המספרים שליליים ($x < y \le 0$)
נכפול במינוס אחד ונקבל $0 \le -y < -x$.
לפי מקרה ב', קיים רציונלי $q'$ כך ש: $-y < q' < -x$.
נכפול חזרה במינוס ונקבל: $x < -q' < y$.
המספר $q = -q'$ הוא הרציונלי המבוקש.
