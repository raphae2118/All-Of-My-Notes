# הגדרה: קטעים וקרניים (3.3.7)

**הגדרה כללית:**
יהיו $a \le b$ שני מספרים ממשיים.
הקטעים הם תתי-קבוצות של הישר הממשי המוגדרים על ידי אי-שוויונים.

## 1. קטעים חסומים (Bounded Intervals)
אלו קטעים שיש להם שני קצוות סופיים. $a$ נקרא הקצה השמאלי, ו-$b$ הקצה הימני.

| סוג הקטע | סימון | הגדרה פורמלית (קבוצה) | הסבר |
| :--- | :--- | :--- | :--- |
| **קטע סגור** | $[a, b]$ | $\{x \in \mathbb{R} : a \le x \le b\}$ | כולל את שני הקצוות. |
| **קטע פתוח** | $(a, b)$ | $\{x \in \mathbb{R} : a < x < b\}$ | לא כולל אף אחד מהקצוות. |
| **חצי פתוח (מימין)** | $[a, b)$ | $\{x \in \mathbb{R} : a \le x < b\}$ | כולל את $a$ אך לא את $b$. |
| **חצי פתוח (משמאל)** | $(a, b]$ | $\{x \in \mathbb{R} : a < x \le b\}$ | כולל את $b$ אך לא את $a$. |

*הערה: קטע נקרא "מנוון" אם נקודות הקצה שלו שוות ($a=b$). במקרה כזה הקטע הסגור הוא נקודה בודדת $\{a\}$, והקטע הפתוח הוא קבוצה ריקה $\emptyset$.*

## 2. קרניים (Rays) - קטעים לא חסומים
קבוצות המוגבלות רק מצד אחד (או לא מוגבלות כלל).

| סוג | סימון | הגדרה פורמלית | הערות |
| :--- | :--- | :--- | :--- |
| **קרן סגורה (משמאל)** | $[a, \infty)$ | $\{x \in \mathbb{R} : x \ge a\}$ | חסומה מלמטה ע"י $a$. |
| **קרן פתוחה (משמאל)** | $(a, \infty)$ | $\{x \in \mathbb{R} : x > a\}$ | |
| **קרן סגורה (מימין)** | $(-\infty, a]$ | $\{x \in \mathbb{R} : x \le a\}$ | חסומה מלמעלה ע"י $a$. |
| **קרן פתוחה (מימין)** | $(-\infty, a)$ | $\{x \in \mathbb{R} : x < a\}$ | |
| **הישר כולו** | $(-\infty, \infty)$ | $\mathbb{R}$ | כל המספרים הממשיים. |
