# אקסיומות הסדר (Order Axioms)

**הגדרה:** השדה $\mathbb{R}$ הוא **שדה סדור**.
משמעות הדבר היא שקיים יחס סדר (מסומן ב- $<$) המקיים את האקסיומות הבאות לכל $a, b, c \in \mathbb{R}$:

## האקסיומות
| שם האקסיומה | ניסוח פורמלי | משמעות |
| :--- | :--- | :--- |
| **O1. טריכוטומיה (Trichotomy)** | לכל זוג $a,b$ מתקיים **בדיוק** אחד מהבאים:<br>$a < b$ או $a = b$ או $b < a$ | תמיד ניתן להשוות בין שני מספרים, ואין מצב של "תיקו" שהוא גם גדול וגם שווה. |
| **O2. טרנזיטיביות (Transitivity)** | אם $a < b$ וגם $b < c$, אז $a < c$ | כלל המעבר: אם א' קטן מ-ב', ו-ב' קטן מ-ג', אז א' קטן מ-ג'. |
| **O3. מונוטוניות החיבור** | אם $a < b$, אז לכל $c$:<br>$a + c < b + c$ | הוספת אותו מספר לשני אגפי אי-השוויון שומרת על הסימן. |
| **O4. מונוטוניות הכפל** | אם $a < b$ וגם **$0 < c$**, אז:<br>$ac < bc$ | כפל במספר **חיובי** שומר על כיוון אי-השוויון. |

---

**הגדרות נגזרות (סימונים):**
1. **גדול מ ($>$):** הביטוי $a > b$ הוא דרך אחרת לכתוב $b < a$.
2. **מספר חיובי:** מספר $x$ מוגדר כחיובי אם $0 < x$.
3. **מספר שלילי:** מספר $x$ מוגדר כשלילי אם $x < 0$.
4. **קטן-שווה ($\le$):** $a \le b$ משמעו $a < b$ **או** $a = b$.

**שים לב ("מוקשים" נפוצים):**
* **כפל במספר שלילי:** האקסיומה O4 מדברת רק על כפל ב-$c$ חיובי. אם כופלים במספר שלילי, סימן אי-השוויון מתהפך ($ac > bc$). זוהי טענה שצריך *להוכיח* (היא לא אקסיומה, אלא מסקנה).
* **שורשים:** לא נובע מכאן אוטומטית שלכל מספר חיובי יש שורש. את זה נצטרך לקבל מאקסיומת השלימות.
